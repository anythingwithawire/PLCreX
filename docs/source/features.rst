.. cli:

Command Line Interface
--------------
PLCreX is a Command Line Interface (CLI) application and can be used as follows:

.. code-block:: console

    (venv) C:\Python\plcrex_project>python -m plcrex [OPTIONS] COMMAND

.. help:

PLCreX Help
----
You can print the help of PLCreX with ``[OPTIONS]`` ``--help``:

.. code-block:: console

    (venv) C:\Python\plcrex_project>python -m plcrex --help

.. version:

PLCreX Version
----
You can print the version of PLCreX with ``[OPTIONS]`` ``--version``/``-v``:

.. code-block:: console

    (venv) C:\Python\plcrex_project>python -m plcrex --version
